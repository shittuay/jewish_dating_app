Our Commitment to Accessibility

Jewish Dating App is committed to ensuring digital accessibility for people of all abilities. We strive to continually improve the user experience for everyone and apply relevant accessibility standards.

Conformance Status

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible for people with disabilities.

Our website is partially conformant with WCAG 2.1 Level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Accessibility Features

We have implemented the following accessibility features:

  • Keyboard navigation support
  • Screen reader compatibility
  • Alternative text for images
  • Clear and consistent navigation
  • Resizable text
  • High contrast mode
  • Focus indicators
  • ARIA labels where appropriate

Known Limitations

While we strive to ensure that our website is accessible, some areas may still present challenges:

  • Some older PDF documents may not be fully accessible
  • Some third-party content may not be fully accessible
  • Some complex interactive features may require additional accessibility improvements

Compatible Technologies

Our website is designed to be compatible with:

  • Recent versions of major browsers (Chrome, Firefox, Safari, Edge)
  • Screen readers (JAWS, NVDA, VoiceOver)
  • Operating system accessibility features
  • Mobile devices and their accessibility features

Accessibility Tools

We provide the following tools to enhance accessibility:

  • Text size adjustment controls
  • High contrast mode toggle
  • Keyboard shortcuts for common actions
  • Skip navigation links

Feedback and Contact

Assessment Methods

We assess our website's accessibility through:

  • Automated testing tools
  • Manual testing by our development team
  • User testing with people who have disabilities
  • Regular accessibility audits

Ongoing Improvements

We are committed to ongoing improvements to our website's accessibility. This includes:

  • Regular accessibility training for our team
  • Periodic accessibility reviews
  • Implementation of new accessibility features
  • Updates based on user feedback

Alternative Access

If you encounter any accessibility issues, we offer alternative ways to access our services:

  • Phone support for account management
  • Email support for all services
  • In-person assistance at our office
  • Alternative format documents upon request
